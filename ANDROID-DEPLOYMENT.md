# Android Deployment Guide

Complete guide to deploy the Self-Hosted File Server on Android using Termux.

---

## 📱 Prerequisites

1. **Android phone** (Android 7.0+)
2. **Termux app** - Install from [F-Droid](https://f-droid.org/packages/com.termux/) (NOT Google Play)
3. **Storage space:** ~200MB for setup + your file storage needs
4. **Internet connection** for initial setup

---

## 🚀 First-Time Setup

### Step 1: Install Termux

1. Download Termux from F-Droid: https://f-droid.org/packages/com.termux/
2. Open Termux
3. Grant storage permissions:
   ```bash
   termux-setup-storage
