# Deployment Scripts for Android

This folder contains scripts for deploying the self-hosted file server on Android using Termux.

## Files

- `android-setup.sh` - First-time setup script for Android/Termux
- `start-server.sh` - Start all services (PostgreSQL, Redis, Backend)
- `stop-server.sh` - Stop all services
- `update.sh` - Update to new version from GitHub release

## Usage

See `../ANDROID-DEPLOYMENT.md` for complete deployment guide.
